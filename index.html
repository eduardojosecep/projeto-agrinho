Sobre o Projeto

"O projeto 'Festejando a Conexão Campo Cidade' é uma celebração das relações entre os mundos rural e urbano. Através deste site, buscamos destacar as trocas culturais, as histórias compartilhadas e a beleza das interações que acontecem quando esses dois universos se encontram, revelando a riqueza que cada um traz para o outro."
